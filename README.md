## OAuth 2.0 چیست؟

OAuth 2.0 یک پروتکل استاندارد برای احراز هویت و مجوزدهی است که به برنامه‌های کاربردی اجازه می‌دهد تا به منابع محافظت‌شده کاربر دسترسی محدود داشته باشند. این پروتکل به طور گسترده برای دسترسی API استفاده می‌شود.

## ویژگی‌های کلیدی

- **امنیت بالا**: استفاده از توکن‌های دسترسی (Access Tokens) به جای اعتبارنامه‌های کاربر
- **انعطاف‌پذیری**: پشتیبانی از چندین نوع Grant Type
- **استاندارد**: پروتکلی استاندارد و پذیرفته شده در صنعت

## Grant Types

OAuth 2.0 از چندین نوع Grant Type پشتیبانی می‌کند:

- **Authorization Code**: برای برنامه‌های سرور-ساید
- **Implicit**: برای برنامه‌های کلاینت-ساید (منسوخ شده)
- **Resource Owner Password Credentials**: برای برنامه‌های قابل اعتماد
- **Client Credentials**: برای دسترسی ماشین به ماشین
